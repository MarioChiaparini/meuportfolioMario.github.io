Projetos que me envolvi

MineAI

Este projeto esta sendo uma das minhas realizações pessoais onde posso finalmente aplicar meus conhecimentos de machine learning. Onde meu modelo de aprendizado de maquina prediz os tipos de minerios analisados na aplicação com base nos dados de espectro Raman.

IoT para filtros de CO2

Este projeto foi uma idealização da IoT Education, onde tive a oportunidade de projetar algoritmos de programação para o sistema embarcado baseado em um ESP32 e uma conexão GSM. Este hardware desenvolvido por meu ex-professor ajudou na automatização do filtro de dioxido de carbono da minha cidade, um dos projetos mais interessantes de IoT que pude presenciar até o momento.

Carro Autonomo (virtual)

Dentro do curso de Stanford (EAD) que estou realizando tenho um projeto de final de curso pelo qual estou fazendo com que um carro (Virtual) se dirija sozinho, juntamente com ferramentas como skitlearn, pandas seaborn e numpy para realização deste projeto.
